# RESTful Web Services with Node.js and Express

This repository is based on the Pluralsight course with the same name as the title by Jonathan Mills. It is used to catch back up on Node.js, since it's been a while since I last used it in a project.

## Importing book data

Inside this project is a file named bookJson.js, which holds the required data for this project to run. You can import this data into MongoDB if you have mongo running. Run the following command to start the import:

```bash
mongo bookAPI booksJson.js
```
